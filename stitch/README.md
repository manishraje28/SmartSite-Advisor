## Design System Tokens — "Titanium Logic"

### Colors
- **Surface (Base):** `#0b1326`
- **Primary:** `#c0c1ff` / Container: `#8083ff` (Indigo)
- **Secondary:** `#4edea3` / Container: `#00a572` (Emerald)
- **Tertiary:** `#bdc2ff` / Container: `#7c87f3`
- **Surface Container High:** `#222a3d` (cards)
- **Surface Container Highest:** `#2d3449` (inputs)
- **Outline Variant:** `#464554` @ 15% (ghost borders)

### Typography
- Font: Inter (all weights)
- Display: 3.5rem Bold, -0.02em tracking
- Headline: 2rem / 1.75rem / 1.5rem Bold
- Body: 1rem / 0.875rem Regular, 1.6 line-height
- Label: 0.75rem / 0.6875rem Medium, Uppercase, 0.05em tracking

### Key Rules
- NO solid borders — use tonal shifts
- Glassmorphism: `white/5%` + `backdrop-blur(12px)`
- Primary buttons: gradient `#6366f1` → `#818cf8`
- Corner radius: 16px
- Dark-mode-first
